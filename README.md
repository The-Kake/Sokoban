# Sokoban
A small game written in C for my studies.

The objective of the game is to place the boxes ($) on the targets (.). You control the Sokoban (@).

How to play?

For Linux users:

Go to your terminal, navigate to the directory where the game is located, then run:

cc main_kaelan_evennou.c -o <name_you_want> -Wall

You can now execute it using:

./<name_you_gave_it>

For Windows users:

Open your terminal, run the "bash" command, then follow the same steps as for the Linux version.

When in the game, to select the level you want to play, type its name and press Enter.

For example, if you want to play level 1, type "niveau1" and press Enter.
